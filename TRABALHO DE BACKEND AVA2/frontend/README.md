# Frontend - Clínica Médica

## Como rodar

1. Instale as dependências:
   ```
npm install
   ```
2. Inicie o servidor de desenvolvimento:
   ```
npm run serve
   ```

Acesse em http://localhost:8080

## Funcionalidades
- Cadastro e login
- Agendamento de consultas
- Consulta de endereço por CEP
- Alerta de chuva
- Painel administrativo
